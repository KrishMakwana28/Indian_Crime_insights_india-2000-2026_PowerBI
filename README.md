## 📌 Project Overview

Crime is a critical issue impacting **public safety, governance, and economic stability**. However, raw crime data alone is difficult to interpret without proper analysis.

This project leverages **Microsoft Power BI** to transform a dataset of **100,000+ crime records and 40+ variables** into a **fully interactive, multi-page analytical dashboard**.

📊 The dashboard enables:

* Exploration of **crime trends over time (2000–2026)**
* Identification of **crime hotspots across India**
* Analysis of **victim & suspect demographics**
* Evaluation of **law enforcement performance**
* Measurement of **financial impact and recovery efficiency**

---

## 🎯 Key Objectives

✔ Analyze crime trends from **2000 to 2026**

✔ Identify **high-crime states and cities**

✔ Study **crime categories and severity distribution**

✔ Understand **victim and suspect behavior patterns**

✔ Evaluate **investigation efficiency and case handling**

✔ Analyze **impact of evidence on case closure**

✔ Measure **financial loss vs recovery gap**

## 🗂️ Dataset Overview

The dataset consists of **100,000+ records** and **40 columns**, grouped into the following categories:

### 🔹 Crime Information

* Crime Type, Severity, Weapon Used, Scope

### 📍 Location Data

* State, City, Police Station, Area Type (Urban/Rural/Semi-Urban)

### 👥 Demographics

* Victim Age, Gender, Count
* Suspect Age, Gender, Repeat Offender Indicator

### 🕵️ Investigation Details

* Arrest Made, Charge Sheet Filed
* Evidence Type (DNA, CCTV, Fingerprints)
* Investigation Priority, Days to Close Case

### 💰 Financial Data

* Property Loss (₹)
* Recovery Value (₹)

## 📊 Key Insights

📉 **Low Recovery Rate**

➡ Only ~25% of total property loss is recovered

⏳ **Investigation Delays**

➡ Average case resolution time exceeds **180 days**

🔁 **High Repeat Offenders**

➡ Nearly **50% of crimes involve repeat offenders**

📹 **Evidence Improves Closure**

➡ Cases with **CCTV/DNA evidence show significantly higher closure rates**
